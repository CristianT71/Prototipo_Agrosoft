# AgroSoft - Sistema de Gestión Agrícola

Sistema web desarrollado para el SENA que facilita la gestión integral de actividades agrícolas, incluyendo cultivos, inventario, ventas y reportes.

## 🌾 Características Principales

### Para Usuarios (Agricultores)
- **Dashboard Interactivo**: Vista general de todas las actividades agrícolas
- **Unidades Productivas**: Gestión de cultivos y parcelas
- **Incidencias**: Registro y seguimiento de plagas y enfermedades
- **Tratamientos**: Control de aplicaciones fitosanitarias
- **Cosechas**: Registro de producción y rendimientos
- **Ventas**: Gestión de comercialización de productos
- **Inventario**: Control de insumos y herramientas
- **Tareas**: Planificación y seguimiento de actividades
- **Alertas**: Notificaciones importantes
- **Reportes**: Análisis y estadísticas de producción
- **Historial**: Trazabilidad completa de operaciones

### Para Administradores
- **Panel Administrativo**: Control total del sistema
- **Gestión de Usuarios**: Administración de permisos y roles
- **Configuración del Sistema**: Personalización de parámetros
- **Reportes Avanzados**: Análisis global de operaciones

## 🚀 Tecnologías

- **Frontend**: React 18.3 + TypeScript
- **Enrutamiento**: React Router 7
- **Estilos**: Tailwind CSS v4
- **Componentes UI**: Radix UI + shadcn/ui
- **Iconos**: Lucide React
- **Animaciones**: Motion (Framer Motion)
- **Formularios**: React Hook Form
- **Gráficos**: Recharts
- **Backend**: Supabase
- **Tipografía**: Inter + Poppins

## 📦 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/CristianT71/agrosoft-sena.git

# Entrar al directorio
cd agrosoft-sena

# Instalar dependencias con pnpm
pnpm install

# Configurar variables de entorno (crear .env.local)
# Ver instrucciones en INSTRUCCIONES_USO.md

# Iniciar servidor de desarrollo
pnpm dev
```

## 🔧 Configuración

1. Crear cuenta en [Supabase](https://supabase.com)
2. Crear nuevo proyecto
3. Configurar las variables de entorno
4. Inicializar las funciones del servidor

Ver documentación completa en `INSTRUCCIONES_USO.md`
